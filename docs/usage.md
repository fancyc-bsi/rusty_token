# Usage

```bash
Rusty Token - JWT Vulnerability Analyzer 1.0
Connor Fancy
Analyzes JWT tokens for common vulnerabilities

USAGE:
    rusty_token [OPTIONS] <token>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -f, --format <format>    The output format of the report (text, json, html) [default: text]

ARGS:
    <token>    The JWT token to analyze

```
